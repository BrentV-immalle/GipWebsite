Foire aux questions

Bienvenue sur mon site Web! Ci-dessous, vous trouvez les questions les plus fréquemment posées sur Microsoft Teams.


Question 1: Comment crée-t-on une nouvelle équipe ?

Pour créer une nouvelle équipe, vous devez suivre les cinq étapes suivantes.

1. D’abord, on clique sur le bouton « Équipes ».
2. Après, on clique sur le bouton « Créer ou rejoindre une équipe ».
3. Ensuite, on clique sur le bouton « Créer une équipe ».
4. Puis, on donne un nom et éventuellement une description à l’équipe. On confirme cela en cliquant sur le bouton « Suivante ».
5. Enfin, on clique sur le bouton « Ignorer ». Vous possédez maintenant votre propre équipe.

Question 2: Comment crée-t-on un nouveau canal pour une équipe ?

Pour créer un nouveau canal pour une équipe, vous devez suivre les cinq étapes suivantes.

1. D’abord, on clique sur le bouton « Équipes ».
2. Après, on clique sur le bouton de l’équipe qu’on veut utiliser.
3. Ensuite, on clique sur le symbole composé de trois points.
4. Puis, on clique sur le bouton « Ajouter un canal ».
5. Enfin, on donne un nom et éventuellement une description au canal. Aussi, on indique l’option « Afficher automatiquement ce canal dans la liste des canaux de tous les membres ». On confirme cela en cliquant sur le bouton « Ajouter ». Vous possédez maintenant un nouveau canal.

Question 3: Comment ajoute-t-on un nouveau membre à une équipe ?

Pour ajouter un nouveau membre à une équipe, vous devez suivre les six étapes suivantes.

1. D’abord, on clique sur le bouton « Équipes ».
2. Après, on clique sur le bouton de l’équipe qu’on veut utiliser.
3. Ensuite, on clique sur le symbole composé de trois points.
4. Puis, on clique sur le bouton « Ajouter un membre ».
5. Après, on tape les noms des membres qu’on veut ajouter à l’équipe. On confirme en cliquant sur le bouton « Ajouter ».
6. Enfin, on clique sur le bouton « Fermer ». Vous avez réussi à ajouter un nouveau membre à l’équipe.

Question 4: Comment charge-t-on des fichiers à partir d’un ordinateur vers un dossier déjà existant ?

Pour charger des fichiers à partir d’un ordinateur vers un dossier déjà existant, vous devez suivre les six étapes suivantes.

1. D’abord, on clique sur le bouton « Équipes ».
2. Après, on clique sur le bouton de l’équipe qu’on veut utiliser.
3. Ensuite, on clique sur le bouton « Fichiers ».
4. Puis, on clique sur le dossier.
5. Après, on clique sur le bouton « Charger » et puis on clique sur le bouton « Fichiers ».
6. Enfin, on double-clique sur les fichiers qu’on veut charger. Vous avez réussi à charger des fichiers.
